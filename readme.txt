Rainfall Prediction System using Machine Learning

Project Overview
This project predicts daily rainfall based on district, month, and day using Machine Learning models. It also includes a Flask web application where users can input values and get rainfall predictions instantly.

Features
-Data preprocessing and cleaning -Transformation of dataset using melt operation -Multiple ML models: Random Forest Linear Regression Support Vector Machine (SVM) -Model comparison using R² score and MSE -Web interface using Flask for prediction -Visualization of model performance

Technologies Used
Python Pandas Scikit-learn Matplotlib Flask

How to Run
1.analyze.py python analyze.py

2.app.py python app.py
